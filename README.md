# 🔫 Cùng tải ngay và chơi

Game bắn súng top‑down 2D được phát triển bằng Unity.  
Người chơi sẽ vượt qua các wave (phase) kẻ thù, nâng cấp vũ khí và kỹ năng để đối đầu với boss cuối cùng.
**Tải ngay game tại đây** - https://drive.google.com/drive/folders/1IiUxp6j5a4AgHxHepMl7uh3-0lZAk9Ty?usp=sharing
- **Hệ thống Wave (Phase) linh hoạt** – Mỗi phase có thể spawn nhiều loại enemy với số lượng tùy chỉnh.
- **Hệ thống Level & XP** – Giết enemy nhận XP, lên level tự động đổi súng (Pistol → SMG → Rifle → Laser → Ultimate).
- **Hệ thống nâng cấp** – Dùng dầu thô để tăng sát thương, băng đạn, tốc độ nạp cho vũ khí.
- **Kỹ năng người chơi** – Dash (Space), Triệu hồi Pet (Q), Bắn năng lượng (E), Dual Wield (R).
- **MiniBoss & FinalBoss** – Boss có nhiều phase và kỹ năng đặc biệt (beam, cầu năng lượng, phun lửa, triệu hồi xúc tu).
- **Đa dạng enemy** – Cận chiến, bắn xa, lăn tấn công, quái nổ, quái hồi máu.
- **Hệ thống UI đầy đủ** – Thanh máu, cooldown kỹ năng, bảng nâng cấp, menu tạm dừng, victory/game over.
- **Âm thanh** – Nhạc nền và hiệu ứng âm thanh (bắn, nhặt dầu, thắng, thua).

## 🛠 Công nghệ sử dụng

- **Engine:** Unity 6.4 (6000.4.2f1)
- **Ngôn ngữ:** C#
- **Thiết kế đồ họa:** Aseprite
- **UI:** TextMeshPro
- **Quản lý phiên bản:** Git & GitHub Desktop
## 🕹 Cách chơi

| Hành động | Phím / Chuột |
|-----------|---------------|
| Di chuyển | WASD |
| Bắn | Chuột trái |
| Nạp đạn | R (hoặc tự động) |
| Dash | Space |
| Kỹ năng Pet | Q |
| Kỹ năng Beam | E |
| Kỹ năng Dual Wield | R |
| Mở bảng nâng cấp | Tab |
| Tạm dừng | ESC |

## 🚀 Cài đặt và chạy với Unity

1. Cài đặt [[Unity Hub](https://unity.com/download)](https://unity.com/fr) 
2. Clone file về máy
3. Open file trong unity
4. Lúc này, unity sẽ thông báo cần cài đúng phiên bản Unity 6.4 (6000.4.2f1), cài đặt phiên bản này hoặc các phiên bản tương tự
5. Mở file đã clone
6. CLick chọn SampleScene và tiếp tục chỉnh sửa
